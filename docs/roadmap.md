# Roadmap

## Milestone 0: Skeleton

- Tauri + Vite + TypeScript project structure.
- Minimal Markdown editing surface.
- English-only project documentation.
- Agent guidance and design document locations.

## Milestone 1: Local Editing

- Native open and save dialogs.
- Autosave drafts.
- Recent documents.
- Basic Markdown preview.
- Keyboard-first command menu.

## Milestone 2: Extension Host

- Plugin manifest format.
- Command registration.
- Editor state API.
- Plugin settings storage.
- Safe enable, disable, and uninstall flows.

## Milestone 3: Collaboration Foundation

- Shared document abstraction.
- Conflict-aware local persistence.
- Optional online sessions.
- Presence and cursor state.
